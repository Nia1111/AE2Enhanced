# AE2Enhanced

> **将无垠的合成时间线折叠于此刻。**

AE2Enhanced 是 **AE2 Unofficial Extended Life (AE2-UEL)** 的终局扩展模组，引入了 **超因果装配枢纽** —— 一座由 344 个方块构成的巨型多方块结构，通过时间折叠批量处理与受控奇点物理，彻底粉碎自动合成中的性能瓶颈。

如果你的 ME 网络在大规模自动合成时卡成幻灯片，这就是答案。

## 核心特性

### 🌌 超因果装配枢纽

一座 344 格巨型多方块，成为 AE 网络中单一、超高效率的合成终点。

| 方块 | 数量 |
|---|---|
| 装配枢纽控制器 | 1 |
| 装配枢纽 ME 接口 | 3 |
| 装配枢纽外壳 | 180 |
| 装配枢纽内壁 | 128 |
| 装配枢纽稳定器 | 32 |

### ⚡ 真正的批量合成

虚拟样板直接在 CPU 库存中批量执行。无需每 tick 推送样板，无卡顿。并行度从 64 起步，配合升级卡可达 **理论无限**。

- **虚拟轨道**：无容器返还的配方瞬间批量完成。
- **真实轨道**：含催化剂、耐久扣减或容器返还的配方被正确处理 —— 催化剂无论批量多大只扣除 1 份，消耗性转换槽位自动逐件处理。

### 🃏 升级模块系统

| 模块 | 效果 | 堆叠上限 |
|---|---|---|
| **时间折叠模块** (并行) | 64 → 2K → 65K → 2M → 67M → ∞ | 5 |
| **时空膨胀模块** (速度) | 每级减半批量冷却，最低 **1 tick** | 5 |
| **维度折叠模块** (扩容) | 每级扩展 480 槽位（5 页）样板存储 | 10 |
| **能量优化模块** (能效) | 降低 AE 网络能耗 | 10 |
| **自动上传模块** | 编码终端完成的 *合成样板* 自动上传至最近的枢纽 | 1 |

### 🌀 黑洞合成

独特的合成系统，利用 **微型奇点** 将物品投入事件视界，转化为异次元时空材料。

**生成仪式**：扔出 64 个 AE2 奇点和 4 个下界之星到地面，然后手持 1 个下界之星右键 ME 控制器。

完整支持 **JEI** 与 **CraftTweaker**：
```zenscript
import mods.ae2enhanced.BlackHole;
BlackHole.addRecipe([<minecraft:stone> * 8, <minecraft:diamond>], <minecraft:obsidian>);
```

## 兼容性

- **CleanroomMC / Kirino**：GL 状态管理已针对现代渲染管线验证。
- **Mixin 驱动**：使用 Late Mixin 安全、无侵入地注入 AE2 合成 CPU 集群。
- **服务端安全**：所有逻辑以服务端为准。客户端只负责渲染漂亮的黑洞。
